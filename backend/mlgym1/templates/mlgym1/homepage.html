This is an Open Source project aiming to provide a user algorithm interaction. Many ML apis do not give us much control over the parameter of the algorithm, we provide a platform were a user can see how an algorithm is behaving and can tweak its parameter to see result change in real time. As Andrew Ng says "without worring about the math" we provide ML algorithms to everyone to play with them. This still require heavy development, we forward to your contribution.
How to use:
  • Choose an algorithm to train.
  • Uploat data points and target variable separately.
  • Provide algorithm parameters.
  • Upload the data wait for training to get finished.
  • See the cost history and get the prediction by uploading new data.
